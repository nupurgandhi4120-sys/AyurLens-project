# AyurLens: Medicinal Plant Companion

[cite_start]AyurLens is an AI-powered platform developed for the **Bharat Academix CodeQuest Hackathon**[cite: 2]. [cite_start]Its mission is to bridge India’s rich AYUSH heritage with modern AI technology, making traditional botanical knowledge accessible, safe, and interactive for everyone[cite: 6].

---

## 💡 Problem Statement
Traditional knowledge regarding medicinal plants is often scattered across multiple sources, leading to several challenges:
* [cite_start]**Identification Gap:** Most people cannot correctly identify medicinal plants[cite: 16].
* [cite_start]**Knowledge Fragmentation:** Information is scattered across multiple sources[cite: 17, 18].
* [cite_start]**Safety Risks:** Misidentification of plants can be risky[cite: 19].
* [cite_start]**Generational Disconnect:** Traditional knowledge is becoming less accessible to younger generations[cite: 20, 23, 24].

---

## 🚀 Key Features
* [cite_start]**Instant AI Identification:** Identify medicinal plants in seconds using advanced image recognition[cite: 31].
* [cite_start]**Verified AYUSH Insights:** Access credible, government-backed information for every plant[cite: 32].
* [cite_start]**Digital Herbal Library:** Browse an extensive database of medicinal uses and precautions[cite: 33].
* [cite_start]**Interactive Learning:** Safely explore botanical knowledge through an AI-powered assistant[cite: 34].

---

## 🛠 Technology Stack
The platform utilizes a modern, scalable tech stack:

* [cite_start]**Frontend:** React 19, JavaScript, Tailwind CSS, HTML 5, and Lucide React[cite: 47, 48, 49, 50, 51, 52].
* [cite_start]**AI & API Integration:** Google Gemini 1.5 Flash API, Plant.id, and Wikipedia[cite: 54, 55, 56, 57].
* [cite_start]**Development & Deployment:** Built with Vite and Node.js [cite: 60, 71][cite_start], and deployed via Netlify[cite: 73].

---

## ⚙️ Core Workflow
The user journey is designed to be seamless and intuitive:



1.  [cite_start]**Open App:** Land on the intuitive, user-friendly homepage[cite: 101, 102].
2.  [cite_start]**Scan the Plant:** Use the camera or upload an image to initiate the identification process[cite: 103].
3.  [cite_start]**Fetch Live Data:** The AI engine processes the image and retrieves relevant data[cite: 104].
4.  [cite_start]**View Results:** Access the plant name and in-depth botanical, safety, and usage details[cite: 105, 106].
